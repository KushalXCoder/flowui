## Styling & Customization

<Callout title="Tailwind CSS" icon={<Paintbrush className="w-4 h-4" />}>
  You can customize the button appearance by modifying the `globals.css` file or updating your Tailwind config.
</Callout>

To change the primary color, update the `--primary` variable in your CSS:

```css title="app/globals.css"
:root {
  --primary: 222.2 47.4% 11.2%; /* Custom HSL value */
  --primary-foreground: 210 40% 98%;
}
```

Or just use standard Tailwind utility classes directly in your components.
