## Highlights

- **Slot-based architecture (Left / Center / Right)**: Intuitive layout positioning.
- **True flex-based center alignment**: Guaranteed centering of the middle slot.
- **Polymorphic as support**: Render any component or HTML element.
- **No enforced viewport logic**: Purely layout-driven, you control responsiveness.
- **Fully composable**: Use only the atoms you need.
- **Router agnostic**: Works seamlessly with any library (Next.js, Remix, etc.).

import { cn } from "@/lib/utils";
import { ComponentPropsWithoutRef, ElementType, ReactNode } from "react";

// Polymorphic component types
type CommonProps = {
    children?: ReactNode;
    className?: string;
};

type PolymorphicProps<T extends ElementType> = {
    as?: T;
} & CommonProps & Omit<ComponentPropsWithoutRef<T>, "as" | "className" | "children">;

/**
 * Main Navbar container.
 */
export const Navbar = <T extends ElementType = "nav">({
    children,
    className,
    as,
    ...props
}: PolymorphicProps<T>) => {
    const Component = as || "nav";
    return (
        <Component
            className={cn(
                "relative flex items-center w-full",
                className
            )}
            {...props}
        >
            {children}
        </Component>
    )
}

/**
 * Left slot of the navbar.
 */
export const NavbarLeft = <T extends ElementType = "div">({
    children,
    className,
    as,
    ...props
}: PolymorphicProps<T>) => {
    const Component = as || "div";
    return (
        <Component
            className={cn(
                "flex items-center gap-2 flex-1 justify-start",
                className
            )}
            {...props}
        >
            {children}
        </Component>
    )
}

## Structure

The `Navbar` uses a flexible slot-based architecture for consistent layouts.

### Alignment Slots
- **`NavbarLeft`**: Flushed left, takes up 1/3 of the space by default to help centering.
- **`NavbarCenter`**: Takes the center 1/3, ensuring children are centered relative to the whole bar.
- **`NavbarRight`**: Flushed right, takes up 1/3 of the space.

### Content Atoms
- **`NavbarLogo`**: Wrapper for branding elements.
- **`NavbarLogoText`**: Styled text for brand names.
- **`NavbarGroup`**: Flex group for horizontal lists.
- **`NavbarItem`**: Individual navigation links or buttons.

## Examples

### Basic Compose

A standard layout with logo, middle links, and a call-to-action.

```tsx
<Navbar className="px-6 py-4 border-b">
    <NavbarLeft>
        <NavbarLogo href="/">
            <div className="w-6 h-6 bg-primary rounded" />
            <NavbarLogoText>FlowUI</NavbarLogoText>
        </NavbarLogo>
    </NavbarLeft>

    <NavbarCenter>
        <NavbarGroup>
            <NavbarItem><a href="#">Features</a></NavbarItem>
            <NavbarItem><a href="#">Pricing</a></NavbarItem>
        </NavbarGroup>
    </NavbarCenter>

    <NavbarRight>
        <button className="...">Sign In</button>
    </NavbarRight>
</Navbar>
```

### Polymorphic Usage

Change the underlying element using the `as` prop.

```tsx
<Navbar as="header">
    <NavbarLogo as="button" onClick={() => ...}>
        Back
    </NavbarLogo>
    
    <NavbarCenter as="div">
        <NavbarItem as="span">Page Title</NavbarItem>
    </NavbarCenter>

    <NavbarRight>
        <NavbarItem as="button">Action</NavbarItem>
    </NavbarRight>
</Navbar>
```

## Props

### Common Props
Available on all components.

| Prop | Type | Default | Description |
| :--- | :--- | :--- | :--- |
| `as` | `ElementType` | Varies | Element to render as. |
| `className` | `string` | - | Custom styling. |
| `children` | `ReactNode` | - | Component content. |

### Default Elements

| Component | Default Element |
| :--- | :--- |
| `Navbar` | `nav` |
| `NavbarLeft` | `div` |
| `NavbarCenter` | `div` |
| `NavbarRight` | `div` |
| `NavbarLogo` | `a` |
| `NavbarLogoText` | `span` |
| `NavbarGroup` | `ul` |
| `NavbarItem` | `li` |
