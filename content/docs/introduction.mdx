## Why Flow UI?

<div className="grid grid-cols-1 md:grid-cols-2 gap-6 my-8">
  <Card title="Design Engineering" icon={<Palette className="text-pink-500" />} href="/docs/theming">
    Meticulously designed with attention to detail. Every pixel counts. Features a robust theming system powered by CSS variables and Tailwind.
  </Card>
  <Card title="Type Safe" icon={<Code className="text-blue-500" />} href="/docs/components/button">
    Built with TypeScript for a superior developer experience. Catch errors at compile time, not runtime.
  </Card>
  <Card title="Accessible" icon={<Book className="text-green-500" />} href="#">
    Follows WAI-ARIA guidelines to ensure your application is usable by everyone. Keyboard navigation and screen reader support out of the box.
  </Card>
  <Card title="Performance" icon={<Zap className="text-amber-500" />} href="#">
    Lightweight and optimized. Zero runtime styles (mostly). Components are tree-shakeable to keep your bundle size small.
  </Card>
</div>
