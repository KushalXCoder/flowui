**Flow UI** is a collection of re-usable components that you can copy and paste into your apps.

It is build as an extension registry for [shadcn/ui](https://ui.shadcn.com), adding new complex components while maintaining the same design philosophy.

## Features

- **Beautiful**: Ultra-modern UI components that look great out of the box.
- **Customizable**: Built with Tailwind CSS, so you can easily customize them to match your brand.
- **Copy & Paste**: You own the code. No npm dependencies for the UI components themselves.

## FAQ

### Why copy/paste?

The idea behind this is to give you full control over the code. You can decide how the components differ or evolve. You can modify the code to your liking.

### Do I need to use Shadcn UI?

Yes. These components are designed to work within the Shadcn UI ecosystem, often reusing base components like `Button`, `Input`, or `Card` from your existing Shadcn installation.

### Is it free?

Yes. Flow UI is open source and free to use for personal and commercial projects.
