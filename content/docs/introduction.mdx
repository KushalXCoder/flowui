## Frequently Asked Questions

<div className="grid gap-6 mt-8">
  <div className="p-4 border rounded-lg bg-card text-card-foreground">
    <h3 className="text-lg font-semibold mb-2 mt-0">Why copy/paste instead of an npm package?</h3>
    <p className="text-muted-foreground m-0">
      The idea behind this is to give you full control over the code. When you install a UI library as an npm package, you're often fighting against the library's abstraction to make small changes. With Flow UI, the code is yours to modify, extend, or strip down as you see fit.
    </p>
  </div>

  <div className="p-4 border rounded-lg bg-card text-card-foreground">
    <h3 className="text-lg font-semibold mb-2 mt-0">Do I need to use Shadcn UI?</h3>
    <p className="text-muted-foreground m-0">
      It is highly recommended. Flow UI is designed to be an **extension registry** for the Shadcn UI ecosystem. We often reuse base components like `Button`, `Input`, or `Card` from your existing Shadcn installation to maintain design consistency.
    </p>
  </div>
  
  <div className="p-4 border rounded-lg bg-card text-card-foreground">
    <h3 className="text-lg font-semibold mb-2 mt-0">Is it free for commercial use?</h3>
    <p className="text-muted-foreground m-0">
      **Yes.** Flow UI is open source and free to use for both personal and commercial projects. You don't need to pay anything or attribute us (though we love it if you do!).
    </p>
  </div>
